# Projet Symfony

Ce projet a été généré avec le squelette Symfony.

## Démarrage rapide

1. Assurez-vous d'avoir PHP et Composer installés.
2. Placez-vous dans le dossier du projet :
   ```powershell
   cd terrabuild
   ```
3. Lancez le serveur de développement Symfony :
   ```powershell
   symfony serve
   ```

## Documentation
Consultez https://symfony.com/doc pour plus d'informations sur le développement avec Symfony.
